Help for Network Management Module 

ADD POP - PointOfPresence.aspx

1.  All field except landline numbers are compulsory.

2. GPDIP (Grid Power Distribution IP)  and BPDIP(BackupPowerDistribution) fields can now be added from this page.

VIEW POP -PointOfPresenceView.aspx

1. In this page, it lists out all the POP registered in the application.

2. Landline column has been removed. GPDBIP & BPDIP columns have been added.

3. From this page, you have the option to

a.deactivate/activate a pop

b. edit pop details

c. delete a pop

EDIT POP - PointOfPresenceEdit.aspx

1. We can go to this page from VIEW POP page to edit the details of a particular POP.

2. GPDBIP & BPDIP columns have been added.

 

MENU changes

1. Seperate Menu link for managing POP have been made available.
